Integration Setup Admin Pages

Although you won't have access to administer the Demo Hub we've setup, we have setup integrations for you. Currently, the Hub supports the following Integration types:

  • Code repositories
  • Docker repositories
  • Kubernetes namespaces
  • Monitoring dashboards
  • Logging dashboards

The Hub allows for multiple integrations of each type to be deployed but for demo purposes we've pre-configured the following:

  • GitHub (code repositories in a demo github org)
  • Quay (docker repositories)
  • A custom Dedicated GKE Kubernetes Cluster for you to create namespaces in
  • Grafana (prometheus) for Monitoring
  • Loki (also Grafana) for Logging
